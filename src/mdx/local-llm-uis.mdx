Sorted by user agents, supported model formats, and programming languages used to build the interface:

- **User agents**: web/desktop/mobile/smartwatch app, browser extension, editor plugin, chat app bots, digital assistant, SMS, email, terminal
- **Model formats**: GGUF, Safetensors (HF, AWQ, GPTQ, ExLlamaV2), MLC, UI-only

Note:

1. "UI-only" entries require a separate install of model backend (e.g. llama.cpp)
2. I'm not sure if this organization makes sense

Please file an [issue](https://github.com/thenameless7741/thenameless.net) or send me a DM on [Twitter](https://twitter.com/thenameless7741) if you have any feedback.
